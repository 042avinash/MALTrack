
MALTrack is a modern Android app for tracking anime and manga with MyAnimeList. It combines MAL account features with extra data from Jikan and AniList to offer a smoother, richer experience for browsing seasonal shows, managing your lists, viewing profiles, reading reviews, following airing countdowns, and getting notified when new episodes release for titles you’re watching.

Built with Kotlin and Jetpack Compose, the app includes MAL OAuth login, anime and manga discovery, detailed title pages, full user lists with search and filters, profile and friends views, customizable defaults, theme/settings persistence, image downloads, and a polished mobile-first UI focused on fast navigation and everyday list management.
